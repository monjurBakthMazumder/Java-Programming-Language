# 📚 Java Programming Language

Welcome to the **Java Programming** repository! 🚀
This repository is designed as a **day-by-day learning path** for beginners, starting from the absolute basics and gradually moving to more advanced topics like recursion and methods. Each day contains **examples, explanations, and practice code** to help you learn Java step by step.

---

## 🔹 Day 01 — Java Fundamentals

Learn the building blocks of Java:

- Introduction to Java
- JDK, JRE, JVM
- Installing Java & Setting Up Environment
- Writing Your First Java Program
- Java Program Structure
- Compilation & Execution Process
- Comments in Java
- Output Methods:
  - `print()`
  - `println()`
  - `printf()`

- Escape Sequences

---

## 🔹 Day 02 — Variables & Data Types

Understand how Java stores and manages data:

- Variables – declaration and initialization
- Naming Rules for Variables
- Primitive Data Types:
  - `int`, `float`, `double`, `char`, `boolean`

- Constants using `final`
- Memory Basics

---

## 🔹 Day 03 — Type Casting & Operators

Learn how to manipulate and convert data:

### Type Casting

- Implicit Casting (automatic)
- Explicit Casting (manual)

### Operators

- Arithmetic, Assignment, Relational, Logical, Unary, Bitwise, Ternary

---

## 🔹 Day 04 — Conditional Statements

Control program flow with decision-making statements:

- `if` Statement
- `if–else` Statement
- Nested `if`
- `else-if` Ladder
- `switch` Statement
- Decision-Making Example Programs

---

## 🔹 Day 05 — Loops

Repeat tasks efficiently using loops:

- `for`, `while`, `do-while` Loops
- Nested Loops
- `break` and `continue` Statements

---

## 🔹 Day 06 — Pattern Printing & Logic Building

Practice logic using loops:

- Number Patterns
- Star Patterns
- Pyramid Patterns
- Reverse Patterns
- Nested Loop Logic

---

## 🔹 Day 07 — Arrays

Learn how to store multiple values in a single variable:

- Array Introduction
- Declaration & Initialization
- Array Memory Concept
- Input & Output
- Array Traversal
- Sum & Average
- Maximum & Minimum
- Linear Search
- Bubble Sort (Basic)
- 2D Arrays & Matrix Input/Output

---

## 🔹 Day 08 — Strings

Work with text in Java:

- String Basics & Declaration
- String Input & Output
- String Methods:
  - `length()`, `charAt()`, `equals()`, `compareTo()`
  - `substring()`, `toUpperCase()`, `toLowerCase()`, `trim()`

- Reverse String
- Palindrome Check
- StringBuilder Basics (mutable strings)

---

## 🔹 Day 09 — Methods (Functions)

Make reusable code blocks:

- Method Syntax & Calling
- Parameters & Arguments
- Return Types
- Method Overloading (Basic Use)
- Passing Arrays to Methods
- Variable Scope (Local & Global)

---

## 🔹 Day 10 — Recursion

Solve problems using methods that call themselves:

- Recursion Concept
- Function Call Stack Idea
- Base Case & Recursive Case
- Factorial Using Recursion
- Fibonacci Series
- Recursive Problem Solving (Array Sum, String Reverse)

---
